Synchronisation entre appareils

Stockez vos données sur un dépôt GitHub privé pour les partager entre plusieurs téléphones.

📖 Comment obtenir le token GitHub ?

1. Créez un dépôt privé sur github.com/new (ex: recolte-data). Cochez « Private » et initialisez avec un README.

2. Allez sur github.com/settings/tokens → « Generate new token (fine-grained) ».

3. Donnez-lui un nom (ex: « app-recolte »), choisissez « Only select repositories » → votre dépôt recolte-data.

4. Dans « Repository permissions », mettez « Contents » sur « Read and write ». Laissez le reste.

5. Générez le token, copiez-le (commence par github_pat_...) et collez-le ci-dessus.

6. Sur le 2ᵉ téléphone : même procédure, même dépôt, même token (ou un nouveau token lié au même dépôt).
